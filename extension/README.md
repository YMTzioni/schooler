# תוסף ייבוא שיעורים ל-Schooler

תוסף Chrome שמייבא פרקים ושיעורים מקובץ JSON (שמיוצא מ-Schooler Course Studio) לדף עריכת הקורס ב-`my.schooler.biz`.

## מה התוסף עושה

לכל **פרק** בקובץ (גרסה 2):

1. לוחץ על **פרק חדש**
2. משנה את שם הפרק (דאבל־קליק)
3. לכל שיעור בפרק:
   - לוחץ על **שיעור חדש**
   - משנה את שם השיעור
   - בוחר **אתר מוטמע**
   - מדביק את קישור ה-`/embed` שלנו ושומר

קבצי גרסה 1 (רק `lessons`) ממשיכים לעבוד בלי יצירת פרקים.

## התקנה ב-Chrome

1. פתח `chrome://extensions`
2. הפעל **Developer mode**
3. לחץ **Load unpacked**
4. בחר את התיקייה `extension` מתוך הפרויקט הזה
5. אחרי עדכון — לחץ **Reload** על התוסף

## זרימת עבודה

1. במערכת שלנו: חלץ פלייליסטים → שמור כל אחד כ**פרק** באותו קורס → **ייצא לתוסף Schooler**
2. פתח בדפדפן את דף עריכת הקורס ב-Schooler (`…/edit`)
3. לחץ על אייקון התוסף
4. טען את קובץ ה-JSON
5. (אופציונלי) הגדר "התחל משיעור מספר" אם רוצים להמשיך מאמצע
6. לחץ **התחל ייבוא**
7. עקוב אחרי הפאנל הצף בפינה; אפשר לעצור בכל רגע

## פורמט JSON (גרסה 2)

```json
{
  "version": 2,
  "exportedAt": "2026-07-12T00:00:00.000Z",
  "course": { "id": "bundle-…", "name": "שם הקורס" },
  "chapters": [
    {
      "order": 1,
      "name": "שם הפרק",
      "playlistId": "PLxxxx",
      "lessons": [
        {
          "order": 1,
          "title": "שם הסרטון מיוטיוב",
          "videoId": "7yA0YU7tVFY",
          "embedUrl": "https://schooler-z7x3.onrender.com/embed/7yA0YU7tVFY"
        }
      ]
    }
  ],
  "lessons": []
}
```

שדה `lessons` השטוח נשמר לתאימות לאחור; התוסף בגרסה 2 עובד לפי `chapters`.

## הערות

- התוסף **מוסיף** פרקים ושיעורים חדשים; הוא לא מוחק קיימים.
- מומלץ להריץ על קורס ריק/נקי לפני ייבוא גדול.
- אם Schooler שינו ממשק, ייתכן שצריך לעדכן סלקטורים ב-`selectors.js`.
