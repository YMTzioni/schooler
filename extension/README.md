# תוסף ייבוא שיעורים ל-Schooler

תוסף Chrome שמייבא שיעורים מקובץ JSON (שמיוצא מ-Schooler Course Studio) לדף עריכת הקורס ב-`my.schooler.biz`.

## מה התוסף עושה

לכל שיעור בקובץ:

1. לוחץ על **שיעור חדש**
2. משנה את שם השיעור (דאבל־קליק)
3. בוחר **אתר מוטמע**
4. מדביק את קישור ה-`/embed` שלנו ושומר

## התקנה ב-Chrome

1. פתח `chrome://extensions`
2. הפעל **Developer mode**
3. לחץ **Load unpacked**
4. בחר את התיקייה `extension` מתוך הפרויקט הזה

## זרימת עבודה

1. במערכת שלנו: ספריית קורסים → **ייצא לתוסף Schooler**
2. פתח בדפדפן את דף עריכת הקורס ב-Schooler (`…/edit`)
3. לחץ על אייקון התוסף
4. טען את קובץ ה-JSON
5. (אופציונלי) הגדר "התחל משיעור מספר" אם רוצים להמשיך מאמצע
6. לחץ **התחל ייבוא**
7. עקוב אחרי הפאנל הצף בפינה; אפשר לעצור בכל רגע

## פורמט JSON

```json
{
  "version": 1,
  "exportedAt": "2026-07-12T00:00:00.000Z",
  "course": { "id": "PLxxxx", "name": "שם הקורס" },
  "lessons": [
    {
      "order": 1,
      "title": "פרק 1: שם השיעור",
      "videoId": "7yA0YU7tVFY",
      "embedUrl": "https://schooler-z7x3.onrender.com/embed/7yA0YU7tVFY"
    }
  ]
}
```

## הערות

- התוסף **מוסיף** שיעורים חדשים; הוא לא מוחק שיעורים קיימים.
- הסדר נקבע לפי `order` מהראשון לאחרון (הראשון נשאר למעלה).
- אם Schooler שינו ממשק, ייתכן שצריך לעדכן סלקטורים ב-`selectors.js`.
- מומלץ להריץ על קורס ריק/נקי לפני ייבוא גדול.
